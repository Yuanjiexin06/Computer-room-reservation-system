# 跟着黑马敲代码

本项目是跟着黑马程序员C++基础视频课程敲出来的代码，主要功能是实现一个简易的机房预约系统，包含学生、老师、管理员三种身份登录，不同身份有不同的操作权限。

## 项目结构

- `include/`：存放项目的头文件。
- `src/`：存放项目的源文件。
- `data/`：存放项目运行时生成的数据文件。
- `README.md`：项目说明文件。
- `LICENSE`：项目许可证文件。
- `.gitignore`：Git忽略文件配置。

## 编译与运行

1. 确保你已经安装了C++编译器（如g++）。
2. 使用以下命令编译项目：

   ```bash
   g++ -o reservation_system src/*.cpp
   ```

3. 运行编译生成的可执行文件：

   ```bash
   ./reservation_system
   ```

## 功能介绍

- 学生可以预约机房、查看自己的预约记录和取消预约。
- 老师可以查看所有学生的预约记录并审核预约。
- 管理员可以添加和查看机房信息。

## 注意事项

- 运行程序前，请确保`data/`目录存在，并且有适当的读写权限。
- 该项目仅用于学习和练习目的，实际应用中请注意数据安全和隐私保护。
